# Billing questions

Use the following resources to get help with billing questions:

- To view billing reports, see [View your billing reports and cost trends](https://docs.cloud.google.com/billing/docs/how-to/reports).

- To learn more about billing, read the
  [Cloud Billing documentation](https://docs.cloud.google.com/billing/docs).

- To resolve billing concerns, use the
  [Google Cloud Billing Troubleshooter](https://support.google.com/cloud/troubleshooter/7279311?hl=en&ref_topic=6288636).

- To request help with billing questions, contact
  [Cloud Billing Support](https://docs.cloud.google.com/support/billing).

- To change or disable billing on a project, go to the **Billing** page in the
  Google Cloud console. For more information, see
  [Modify a project's billing settings](https://docs.cloud.google.com/billing/docs/how-to/modify-project).

  > [!CAUTION]
  > **Caution:** When you disable billing for a project, you also disable all product resources in that project.
