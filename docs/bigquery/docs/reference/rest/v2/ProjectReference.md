# ProjectReference

- [JSON representation](https://docs.cloud.google.com/bigquery/docs/reference/rest/v2/ProjectReference#SCHEMA_REPRESENTATION)

A unique reference to a project.

| JSON representation |
|---|
| ``` { "projectId": string } ``` |

| Fields ||
|---|---|
| `projectId` | `string` Required. ID of the project. Can be either the numeric ID or the assigned ID of the project. |
