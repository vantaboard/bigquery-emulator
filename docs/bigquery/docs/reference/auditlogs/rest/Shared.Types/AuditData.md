## Table

Describes a BigQuery table. See the [Table](https://docs.cloud.google.com/bigquery/docs/reference/v2/tables) API resource for more details on individual fields. Note: `Table.schema` has been deprecated in favor of `Table.schemaJson`. `Table.schema` may continue to be present in your logs during this transition.

| JSON representation |
|---|
| ``` { "tableName": { object (`https://docs.cloud.google.com/bigquery/docs/reference/auditlogs/rest/Shared.Types/AuditData#TableName`) }, "info": { object (`https://docs.cloud.google.com/bigquery/docs/reference/auditlogs/rest/Shared.Types/AuditData#TableInfo`) }, "schemaJson": string, "view": { object (`https://docs.cloud.google.com/bigquery/docs/reference/auditlogs/rest/Shared.Types/AuditData#TableViewDefinition`) }, "expireTime": string, "createTime": string, "truncateTime": string, "updateTime": string, "encryption": { object (`https://docs.cloud.google.com/bigquery/docs/reference/auditlogs/rest/Shared.Types/AuditData#EncryptionInfo`) } } ``` |

| Fields ||
|---|---|
| `tableName` | ``object (`https://docs.cloud.google.com/bigquery/docs/reference/auditlogs/rest/Shared.Types/AuditData#TableName`)`` The name of the table. |
| `info` | ``object (`https://docs.cloud.google.com/bigquery/docs/reference/auditlogs/rest/Shared.Types/AuditData#TableInfo`)`` User-provided metadata for the table. |
| `schemaJson` | `string` A JSON representation of the table's schema. |
| `view` | ``object (`https://docs.cloud.google.com/bigquery/docs/reference/auditlogs/rest/Shared.Types/AuditData#TableViewDefinition`)`` If present, this is a virtual table defined by a SQL query. |
| `expireTime` | ``string (`https://protobuf.dev/reference/protobuf/google.protobuf#timestamp` format)`` The expiration date for the table, after which the table is deleted and the storage reclaimed. If not present, the table persists indefinitely. Uses RFC 3339, where generated output will always be Z-normalized and use 0, 3, 6 or 9 fractional digits. Offsets other than "Z" are also accepted. Examples: `"2014-10-02T15:01:23Z"`, `"2014-10-02T15:01:23.045123456Z"` or `"2014-10-02T15:01:23+05:30"`. |
| `createTime` | ``string (`https://protobuf.dev/reference/protobuf/google.protobuf#timestamp` format)`` The time the table was created. Uses RFC 3339, where generated output will always be Z-normalized and use 0, 3, 6 or 9 fractional digits. Offsets other than "Z" are also accepted. Examples: `"2014-10-02T15:01:23Z"`, `"2014-10-02T15:01:23.045123456Z"` or `"2014-10-02T15:01:23+05:30"`. |
| `truncateTime` | ``string (`https://protobuf.dev/reference/protobuf/google.protobuf#timestamp` format)`` The time the table was last truncated by an operation with a `writeDisposition` of `WRITE_TRUNCATE`. Uses RFC 3339, where generated output will always be Z-normalized and use 0, 3, 6 or 9 fractional digits. Offsets other than "Z" are also accepted. Examples: `"2014-10-02T15:01:23Z"`, `"2014-10-02T15:01:23.045123456Z"` or `"2014-10-02T15:01:23+05:30"`. |
| `updateTime` | ``string (`https://protobuf.dev/reference/protobuf/google.protobuf#timestamp` format)`` The time the table was last modified. Uses RFC 3339, where generated output will always be Z-normalized and use 0, 3, 6 or 9 fractional digits. Offsets other than "Z" are also accepted. Examples: `"2014-10-02T15:01:23Z"`, `"2014-10-02T15:01:23.045123456Z"` or `"2014-10-02T15:01:23+05:30"`. |
| `encryption` | ``object (`https://docs.cloud.google.com/bigquery/docs/reference/auditlogs/rest/Shared.Types/AuditData#EncryptionInfo`)`` The table encryption information. Set when non-default encryption is used. |

## TableInfo

User-provided metadata for a table.

| JSON representation |
|---|
| ``` { "friendlyName": string, "description": string, "labels": { string: string, ... } } ``` |

| Fields ||
|---|---|
| `friendlyName` | `string` A short name for the table, such as `"Analytics Data - Jan 2011"`. |
| `description` | `string` A long description, perhaps several paragraphs, describing the table contents in detail. |
| `labels` | `map (key: string, value: string)` Labels provided for the table. An object containing a list of `"key": value` pairs. Example: `{ "name": "wrench", "mass": "1.3kg", "count": "3" }`. |
