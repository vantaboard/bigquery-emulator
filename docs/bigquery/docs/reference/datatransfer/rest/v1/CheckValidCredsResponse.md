# CheckValidCredsResponse

- [JSON representation](https://docs.cloud.google.com/bigquery/docs/reference/datatransfer/rest/v1/CheckValidCredsResponse#SCHEMA_REPRESENTATION)

A response indicating whether the credentials exist and are valid.

| JSON representation |
|---|
| ``` { "hasValidCreds": boolean } ``` |

| Fields ||
|---|---|
| `hasValidCreds` | `boolean` If set to `true`, the credentials exist and are valid. |
